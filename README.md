# 📚 BookVerse – MERN Multi-Vendor Book Store

BookVerse is a full-stack MERN Book Store application developed as part of the SkillWallet Full Stack MERN Stack Program.

The platform supports three different roles:

- 👤 User
- 🏪 Seller
- 👑 Admin

Users can browse books, place orders, and track deliveries.

Sellers manage inventory, accept/reject orders, and update order status.

Admins manage books, sellers, users, and platform statistics.

# ⚙ Installation

Clone Repository

```bash
git clone https://github.com/BoodiPuneethReddy/BookVerse-SkillsWallet.git
```

Install Frontend

```bash
cd client
npm install
npm run dev
```

Install Backend

```bash
cd server
npm install
npm run dev
```
